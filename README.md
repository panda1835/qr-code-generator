# Free QR Code Generator

I got tired of QR code generators that hide basic features behind paid plans, signups, ads, tracking pages, or unnecessary requirements.

So I made this one.

No signup. No ads. No account. No paid gate. Just enter a link or text, generate a QR code, and download it.

## What It Does

- Generates QR codes for URLs or plain text
- Lets you download the QR code as a PNG
- Works without creating an account
- Does not add redirect links to the QR code destination

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
